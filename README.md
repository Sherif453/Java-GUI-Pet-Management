Pet Health Management System

A simple Java GUI application for managing and tracking pet health information. Users can register pets, schedule appointments, track medications, and store health notes through an intuitive interface. The project demonstrates Java OOP principles, JavaFX GUI design, and clean modular structure.

Features
- Add and manage pet profiles  
- Schedule and view vet appointments  
- Track medications and dosages  
- Store general health notes  
- Login and registration system  
- Multiple FXML-based screens with controllers

Technologies Used
- Java  
- JavaFX  
- FXML  
- OOP principles  
- NetBeans IDE  

How to Run
1. Clone the repository: https://github.com/Sherif453/Java-GUI-Pet-Management
2. Open the project in IntelliJ, Eclipse, or NetBeans
3. Make sure JavaFX libraries are correctly configured
4. Run the main application file
